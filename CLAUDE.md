# Minji Kim — Personal Brand Website

## Project Overview
A personal brand website for Minji Kim. Includes a portfolio, blog, and AI powered chatbot that answers questions about my background.

## Tech Stack
- **Language:** TypeScript
- **Frontend:** Next.js + TailwindCSS
- **Backend:** Supabase (backend + database, Postgres SQL)
- **Deployment:** Vercel

## Repo Structure
- Parent repo: `minji-web`
- Frontend repo: `minji-web-frontend`
- Backend repo: `minji-web-backend`

## What the Site Includes (Tentative V1)
- Who I am and what I do
- Work experience and projects
- Blog — my POV on operations and scaling
- AI chatbot powered by Claude API that answers questions about my background
- Contact section

## Constraints
- V1 only — keep it simple
- No over-engineering
- Mobile friendly
- Do not add features outside of what is listed above
